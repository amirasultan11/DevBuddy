# Sound Assets for Kids Coding Game

## Required Sound Files

Place the following `.mp3` files in this directory (`assets/sounds/`):

### 1. `click.mp3`
- **When**: User taps a direction button (Up/Down/Left/Right)
- **Suggested Sound**: Short, playful "pop" or "click" sound
- **Duration**: ~100-200ms
- **Example**: Bubble pop, button click, soft beep

### 2. `move.mp3`
- **When**: Hero moves to a new grid cell during execution
- **Suggested Sound**: Quick "jump" or "hop" sound
- **Duration**: ~200-300ms
- **Example**: Cartoon jump, soft thud, whoosh

### 3. `win.mp3`
- **When**: Hero reaches the star (goal)
- **Suggested Sound**: Cheerful celebration music/sound
- **Duration**: ~2-3 seconds
- **Example**: Victory fanfare, kids cheering, happy jingle

### 4. `fail.mp3`
- **When**: Hero hits the rock (obstacle)
- **Suggested Sound**: Gentle "bonk" or "buzz" sound (not scary!)
- **Duration**: ~500ms-1s
- **Example**: Cartoon bonk, "uh-oh" sound, gentle buzzer

## Free Sound Resources

You can find free sound effects at:
- **Freesound.org**: https://freesound.org/
- **Zapsplat**: https://www.zapsplat.com/
- **Mixkit**: https://mixkit.co/free-sound-effects/
- **Pixabay**: https://pixabay.com/sound-effects/

## File Format
- Format: MP3
- Sample Rate: 44.1kHz recommended
- Bit Rate: 128kbps or higher

## Testing
The game will work without these files (sounds will be silently skipped), but adding them greatly enhances the experience for kids!
