#  DevBuddy
Your AI-Powered Tech Career Compass & Learning Hub

##  Overview
DevBuddy is an innovative EdTech platform designed to eliminate the "where to start" confusion for aspiring tech professionals and kids
It acts as a personalized career compass, combining psychological profiling (16 Personalities) with AI to generate customized, gamified learning roadmaps 

Currently participating in the Mans Tech Hackathon

##  Core Features
*  The Compass (AI-Driven Guidance): Evaluates users through the 16 Personalities framework and skill assessments to recommend the perfect tech track (e.g., Data Analysis, Mobile Dev, UI/UX)
*  Gamified Learning Roadmaps: Step-by-step paths where users must complete specific tasks and pass end-of-stage quizzes to unlock the next level
*  Kids Tech Track: A dedicated, simplified, and highly engaging roadmap tailored specifically for younger learners stepping into the programming world
*  The Hub (Opportunities Center): A centralized dashboard featuring external learning resources, local/global training grants, and internships to bridge the gap between learning and the job market

##  Tech Stack & Architecture (Under Development)
* Frontend: Flutter (Dart) for a seamless, cross-platform mobile experience
* Backend & Content Management: Headless CMS (Planned) to easily manage dynamic content like quizzes, roadmaps, and grants without hardcoding
* Intelligence: Integration with AI APIs (e.g., Gemini/OpenAI) for dynamic roadmap generation and automated quiz creation based on user progression

##  Current Status
MVP Phase: We are currently finalizing the core architecture, UI/UX flows, and the integration mapping for AI features and Headless CMS
